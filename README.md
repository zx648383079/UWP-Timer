 ## 使用

 请配置信息

`UWP-Timer\Repositories\Constants.cs`

```c#
public const string ApiEndpoint = ""; // api 入口
public const string AppId = "<APP ID>"; // api 验证用的 appid
public const string Secret = "<APP SECRET>"; // api 验证用的 app secret
```
